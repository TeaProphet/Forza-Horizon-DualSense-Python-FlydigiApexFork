# 🏎️ Forza Horizon - Адаптивные триггеры DualSense (Форк для Flydigi Apex 4/5)

Этот репозиторий является форком оригинального проекта [Forza-Horizon-DualSense-Python](https://github.com/HamzaYslmn/Forza-Horizon-DualSense-Python), измененным для добавления полной поддержки вибрации (telemetry rumble) для контроллеров **Flydigi Apex 4/5** в режиме эмуляции DualSense.

> [!WARNING]
> Это неофициальный форк, протестированный только на контроллере **Flydigi Apex 4** под управлением **Windows 11**. Другие модели, типы подключения или операционные системы могут работать, но не были официально проверены.

---

## 🛠️ Что исправлено и добавлено в этом форке

1. **Вибрация Flydigi Apex 4/5**: Прошивка эмуляции DualSense на Apex 4/5 сбрасывает значения вибрации, если эффекты триггеров отправляются в том же пакете. Мы решили это разделением HID-записей на два последовательных пакета: один для триггеров, другой для моторов вибрации.
2. **Фильтрация вибрации на асфальте**: Добавлена мертвая зона покрытия (`rumble_surface_deadzone` со значением по умолчанию `0.05`) для устранения постоянного гула на чистом асфальте. Теперь вибрация ощущается только при съезде на бездорожье (грязь, гравий, трава, песок).
3. **Вибрация при потере сцепления**: Добавлена мертвая зона пробуксовки (`rumble_slip_deadzone` со значением по умолчанию `0.10`), чтобы отсечь незначительное проскальзывание при обычных поворотах. Вибрация срабатывает только при реальной потере сцепления (пробуксовка на старте, заносы или дрифт).
4. **Поребрики и стыки**: Столкновения с поребриками отправляют отчетливые высокочастотные импульсы напрямую на легкий правый мотор.
5. **Интерактивная настройка на лету**: Вкладка настроек TUI содержит новый блок **Telemetry Rumble (Flydigi Apex 4/5)** для регулировки всех масштабов и мертвых зон в реальном времени.
6. **Исправление кнопки проверки вибрации**: Исправлена кнопка "Test Rumble" в TUI. Теперь проверочные вибрации не сбрасываются мгновенно в 0 циклом телеметрии.

## 🎮 Эмуляция Flydigi DualSense

Flydigi Space Station требует запуска поддерживаемой игры (например, *Marvel's Spider-Man: Miles Morales*) для перевода контроллера в режим эмуляции DualSense.

**Этот проект полностью автоматизирует данный процесс!**
При запуске программа автоматически копирует системный исполняемый файл и запускает скрытый фоновый процесс под именем `MilesMorales.exe`. Это трюк (хитрость) для обмана Flydigi Space Station, чтобы заставить ее сразу включить режим эмуляции DualSense. При выходе из программы фоновый процесс автоматически закрывается.

Никаких ручных действий, переименований или отдельных запусков не требуется! (Эту функцию можно отключить, сняв галочку `Auto-activate Flydigi DualSense emulation mode` в настройках).

## ⚙️ Настройка в игре

В Forza Horizon откройте **Настройки -> HUD и игровой процесс** и прокрутите в самый низ:

- **Вывод данных (Data Out)**: ВКЛ (ON)
- **IP-адрес вывода данных**: 127.0.0.1
- **Порт вывода данных**: 5300

## 🚀 Инструкции по установке и запуску

### Системные требования
- Windows 10/11 или Linux.
- Подключенный контроллер Flydigi Apex 4/5.
- Перед запуском утилиты необходимо запустить **Flydigi Space Station 3/4**.
- Внутри Flydigi Space Station перейдите в раздел **Adaptive Trigger (Адаптивные триггеры)** -> выберите профиль **Marvel's Spider-Man: Miles Morales** -> измените режим триггера на **DS Mode (Режим DualSense)** -> установите режим **Adaptive (Адаптивный)** в положение **ON (ВКЛ)**.

### Как запустить (с автоматической сборкой)
Для локальной сборки проекта из исходников без скачивания старых версий с GitHub:
1. Скачайте этот репозиторий.
2. Дважды кликните по файлу **`build_and_start.bat`** в корневой папке.
3. Скрипт проверит наличие диспетчера пакетов `uv` (установит его при необходимости), подтянет все зависимости, соберет локальный пакет и запустит интерфейс TUI.

### Запуск из исходного кода (для разработчиков)
Если вы хотите запускать сырые Python-файлы напрямую:
1. Перейдите в папку `src/`.
2. Установите зависимости и запустите проект через `uv`:
   ```powershell
   uv sync
   uv run main.py
   ```
